## Overview

## Getting Started

1. Install dependencies:

```bash
pnpm install
```

2. Start the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Framework**: Next.js 16+ (React 19)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Linting**: ESLint
